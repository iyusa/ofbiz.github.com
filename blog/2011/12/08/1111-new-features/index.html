2011/11 OFBiz 新功能

Dec 8th, 2011

請參考 最新功能 網頁。

  • 更新 jQuery 到版本 1.7, 且 jQuery UI 程式庫到版本 1.8.16
  • 更新到 Tomcat 7

  • 更新 Jackrabbit 程式庫到版本 2.3.3
  • 增加查詢支援, 能用 SQL2 / JQOM 查詢語法來查儲存庫
